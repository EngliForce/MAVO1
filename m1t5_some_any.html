🚨 MISSION: SOME VS ANY

Hoeveel heb je ergens van? In het Engels gebruik je 'some' of 'any'. De keuze hangt bijna altijd af van het soort zin dat je maakt!

SOME
Bevestigende zinnen (+).

Gebruik dit als je zegt dat er iets wel is.
• I have some water.
• There are some biscuits.
• She bought some fruit.
ANY
Ontkenningen (-) & Vragen (?).

Gebruik dit als er niets is of als je iets vraagt.
• I don't have any money.
• Are there any crisps?
• We haven't got any milk.
🚀 PRO-TIP: DE UITZONDERING
Soms gebruik je SOME in een vraag, maar alleen als je iets aanbiedt of ergens om vraagt:
  • Would you like some tea? (Aanbod)
  • Can I have some sugar, please? (Verzoek)

Quick Scan Checklist

✅ SOME: Positieve zinnen + Aanbiedingen.

✅ ANY: Zinnen met 'not', 'n't', 'never' + Normale vragen.

✅ LET OP: Beide woorden gebruik je voor meervoud of ontelbare woorden.
